# **App Name**: Pantanal Condomínio Manager

## Core Features:

- Supabase Authentication: Enable user authentication using Supabase Auth for secure access.
- Fine Management: Track and manage fines issued to residents, storing data in Supabase's PostgreSQL database.
- Gourmet Area Reservations: Allow residents to reserve the gourmet area with real-time availability updates via Supabase Realtime.
- User Phone Management: Allow users to register their phone for administrative staff
- Real-time Notifications: Display real-time updates for fines, reservations, and community announcements using Supabase Realtime.

## Style Guidelines:

- Primary color:  (###004db6) evoking tranquility and reliability.
- Background color: Light grey (#F0F4F8), providing a clean and neutral base.
- Accent color: Muted green (##ffc309), symbolizing growth and harmony.
- Body and headline font: 'PT Sans', a humanist sans-serif that combines a modern look and a little warmth.
- Use clean and modern icons representing fines, reservations, and user profiles.
- Implement a clear, intuitive layout optimized for easy navigation and information access.
- Incorporate subtle animations for transitions and feedback, enhancing the user experience.
